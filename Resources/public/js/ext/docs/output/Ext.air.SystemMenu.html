Class Ext.air.SystemMenu

Package:Ext.air
Defined In:SystemMenu.js
Class:SystemMenu
Extends:Object
Provides platform independent handling of adding item to the application menu, creating the menu or items as needed.

This class also provides the ability to bind standard Ext.Action instances with NativeMenuItems

This class is a singleton and cannot be created directly.
